Table Component

O componente Table é utilizado para exibir dados tabulares com suporte a ordenação, paginação e seleção de linhas.

Exemplo 1: Tabela dinâmica com colunas, tipos e ações.

Propriedades

Propriedade Tipo Obrigatório? Descrição Exemplo
columns TableColumn[] Sim Array de objetos que define as colunas da tabela. Cada coluna pode ter título, propriedade, largura, ordenação e ações. 
{{columnsSnippet}}
data any[] Sim Array de objetos que representa as linhas da tabela. As chaves devem corresponder ao prop das colunas. 
{{dataSnippet}}
columnTypes {{columnTypesSnippet}} Não Define o tipo de exibição para cada coluna. Use tag para badges, date para datas, time para horas. 
{{columnTypesExampleSnippet}}
statusClass
(ou <prop>Class)		 string Não Permite customizar a cor da badge para colunas do tipo tag. Use classes Bootstrap: text-bg-primary, text-bg-success, etc. 
{{statusClassSnippet}}
actions TableAction[] Não Define ações para a coluna. Cada ação pode ter title, icon (Material Icons) e callback (função chamada ao clicar). 
{{actionsSnippet}}

Dicas de uso

  • Para badges coloridas, defina <prop>Class no seu objeto de dados, exemplo: statusClass: 'text-bg-success'.
  • Para datas, envie um objeto Date ou string compatível e defina columnTypes como date.
  • Para ações, defina a coluna actions e passe um array de ações com callback.
  • Ícones das ações usam Material Icons.
