Breadcrumb Component

O componente Breadcrumb é utilizado para exibir a navegação hierárquica do usuário dentro da aplicação, permitindo fácil retorno a páginas anteriores.

Este é um exemplo de breadcrumb simples com três níveis de navegação. O usuário pode clicar em "Home" ou "Seção" para navegar para essas páginas, enquanto "Página Atual" indica a página em que o usuário está atualmente.

Propriedades

items Array<{{ '{' }} label: string, url?: string {{ '}' }}> [] Array de objetos que define os itens do breadcrumb. Cada item deve ter um rótulo e pode ter uma URL opcional para navegação.
