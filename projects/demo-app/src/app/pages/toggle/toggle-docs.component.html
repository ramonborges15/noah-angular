O componente Toggle é utilizado para criar um switch estilizado com Bootstrap, integrando com Reactive Forms.

Propriedades

formControlName string - Permite integração com Reactive Forms.
habilitarFiltro boolean false Exibe o título ao lado do toggle se for true.
title string '' Título exibido ao lado do toggle quando habilitarFiltro for true.
