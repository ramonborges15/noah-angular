Checkbox Component

O componente Checkbox é utilizado para criar caixas de seleção (checkboxes) ou botões de opção (radio buttons) estilizados com Bootstrap e tipografia Poppins.

Seleção múltipla
Permite selecionar várias opções ao mesmo tempo.
Selecionado: {{ multiSelected | json }}
Seleção única
Permite selecionar apenas uma opção (comportamento de radio button).
Selecionado: {{ singleSelected | json }}
Checkbox desabilitado
Todas as opções estão desabilitadas e não podem ser alteradas.
Com validação obrigatória
Este campo é obrigatório. Clique no botão para forçar a exibição do erro se nada estiver selecionado.

Propriedades

options Array<{{'{'}} label: string, value: any {{'}'}}> [] Array de objetos que define as opções exibidas. Cada item deve ter um rótulo (label) e um valor (value).
multiple boolean false Define se o componente permite múltipla seleção (true) ou apenas uma opção (false).
control FormControl - FormControl do Reactive Forms utilizado para controlar o valor, estado de desabilitado e validação do componente.
selectedChange EventEmitter<any[] | any> - Evento emitido sempre que o valor selecionado é alterado.
