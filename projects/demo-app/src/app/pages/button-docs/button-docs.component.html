Button Component

O componente Button é utilizado para criar botões estilizados com Bootstrap e tipografia Poppins.

Este é um exemplo de botão desabilitado. O atributo [disabled]=" true" faz com que o botão fique inativo, impedindo qualquer interação do usuário.

Este é um exemplo de botão habilitado. O botão pode ser clicado e executará a ação associada.

Propriedades

title string 'Título do botão' Título exibido no botão
color string '#ffffff' Cor do texto do botão
backgroundColor string '#660ea5' Cor de fundo do botão
borderColor string '#660ea5' Cor da borda do botão
minWidth string '100px' Largura mínima do botão
disabled boolean false Indica se o botão está desabilitado
loading boolean false Indica se o botão está em estado de carregamento
loadingText string 'Carregando...' Texto exibido quando o botão está em estado de carregamento
